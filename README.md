Este é um projeto de extensão para servidores de Minecraft (Spigot/Paper) desenvolvido com o objetivo de criar um sistema de recuperação de itens após a morte do jogador. Inspirado em mecânicas como o Keep Inventory, este plugin oferece uma alternativa mais imersiva ao evitar a perda permanente de itens. acesse a playlist [aqui](https://www.youtube.com/watch?v=Njn4qSDnTJc&list=PL9ZxYIDQ9iTOwEnCXBOgR5os6OopHdb_8&pp=sAgC)
